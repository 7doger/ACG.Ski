# ACG.Ski
服务于国内ACG爱好者与相关开源项目的免费子域名

## 申请条件（满足其中一项即可）
- ACG爱好者的博客
- ACG相关的开源、托管项目
- ACG人物、文化相关的站点


## 申请说明
1. Fork 本项目
2. 将域名信息添加到 [CNAME 绑定](https://github.com/7doger/acg.ski/blob/master/main/active_cname.js)
3. 提交 Pull Request
4. 等待审核，确认后部署

## 支持的服务
- GitHub Pages
- GitLab Pages
- Vercel
- CloudFlare Pages
